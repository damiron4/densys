# Densys
Software Engineering Project

## Project Description
For this project, we worked with PERN stack: PostgreSQL, Express, React.js and Node.js.

## Installation

To run the client-side of the project:
```
cd client
npm start 
```

To run the server call:
```
cd server
node index
```
or
```
cd server
nodemon index
```
